# jxl-oxide
JPEG XL decoder written in pure Rust.

## Supported features
- Simple Modular and VarDCT images
- Progressive LF (LF frames)
- XYB
- HDR images
- Chroma subsampled YCbCr
- Restoration filters
- Image features
  - Patches
  - Splines
- Frame composition
- Animation
- Cropped decoding (sort of)
